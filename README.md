# Daily Algorithms Practice 🧠

A collection of daily algorithm challenges and data structure implementations in TypeScript and Python. This repository serves as a personal log for mastering computer science fundamentals and solving LeetCode problems.

## 📂 Project Structure
- `data-structures/`: Core data structure implementations (Linked Lists, Trees, Graphs, etc.)
- `solutions/`: Problem-solving implementations categorized by language.
  - `typescript/`: Solutions in TypeScript.
  - `python/`: Solutions in Python.
- `leetcode/`: Specific solutions for LeetCode problems.

## 🚀 Getting Started
Clone the repository:
```bash
git clone https://github.com/shentian-AwA/daily-algorithms-practice.git
```
